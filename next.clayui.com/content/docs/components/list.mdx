## Variants and Best Pratices

A list can composable by a Header, an Item and Quick Actions.

<List />

### Header

Use `Header` composition to define a Header section on List.

<ListHeader />

### Item

Use `Item` composition to define Items on List.

<ListItem />

### ItemField

Use `ItemField` inside `Item` composition for wrapping anything inside `Item`.

Use [`expand`](#api-expand) property to indicate if field should automatically expand its width to fit in the containing space.

<ListItemField />

### ItemText

If you want to define texts inside the `Item`, use `ItemText` variant.

Also, you can use this component to define subtexts, use [`subtext`](#api-subtext) property.

<ListItemText />

### ItemTitle

Use `ItemTitle` composition for defining titles.

<ListItemTitle />

### QuickActionMenu

Use `QuickActionMenu` composition inside `Item` for defining a QuickActionMenu. 

Wrap `QuickActionMenu.Item` inside `QuickActionMenu` for defining an `Item` of QuickActionMenu. See the following example:

<ListQuickActionsMenu />
