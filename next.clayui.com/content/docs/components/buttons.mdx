## Variants and Best Pratices

Button component can be used as a Button or a Button Group using [`ClayButton.Group`](#group) variant as low-level API.

### Default

<ButtonWithStar />

#### Display Types

You can determine how your button can be displayed using [`displayType`](#api-displayType) property:

Use [`unstyled`](#api-unstyled) to reset all the stylings from Bootstrap 4.

If you want use the button as a link use [`link`](#api-link).

<ButtonDisplayTypes />

#### Block

Use [`block`](#api-block) property to use your button as a block element.

<ButtonBlock />

#### Monospaced

Use [`monospaced`](#api-monospaced) to make your button monospaced.

<ButtonMonospaced />

#### Small

If you want a smaller button, use [`small`](#api-small) property.

<ButtonSmall />

### Group

You can use the variant `ClayButton.Group` for creating button groups:

Use [`spaced`](#api-spaced) property to create spacing between buttons.

<ButtonGroup />
